# NUK App 2

## Introduction
NUK App 2 是一個非官方的高雄大學校園資訊平台的應用程式。該專案以開源的形式進行，旨在提供一個方便、快捷的方式讓高雄大學的學生能夠獲取校園相關的資訊。本文將介紹該應用程式的功能，如果你對這個專案有興趣想要貢獻其他功能，也可以發PR給我，大感謝!  
**目前還在開發初期，大部分之前一代的功能都還未完成，也有許多問題若不幸遇到BUG也可發ISSUE給我喔!**

## Features
NUK App 2 提供以下功能：
- 課程查詢 (資料目前是可涵蓋碩班)
- 模擬選課
- .....
